# Запуск и открытие локальной версии сайта в браузере

Когда пользователь просит "Запусти сайт", "Открой сайт" или вызывает этот воркфлоу, выполни шаг открытия сервера разработки Vite.

// turbo-all
1. В терминале выполни команду `open http://localhost:5173/planner-park/` (стандартный адрес Vite с учетом нашего базового пути).
2. Сообщи пользователю, что локальная версия проекта была успешно открыта в браузере по умолчанию.
