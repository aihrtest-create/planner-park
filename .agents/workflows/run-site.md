# Запуск и открытие локальной версии сайта в браузере

Когда пользователь просит "Запусти сайт", "Открой сайт" или вызывает этот воркфлоу, выполни шаги по запуску сервера разработки Vite и открытию его в браузере.

// turbo-all
1. Проверь, запущен ли локальный сервер (например, с помощью команды `lsof -i :5173` или попытавшись сделать HTTP-запрос).
2. Если сервер не запущен, выполни команду `npm run dev` в persistent терминале (в корне проекта) для его запуска в фоне и подожди, пока он запустится (пару секунд).
3. В терминале выполни команду `open http://localhost:5173/planner-park/` (стандартный адрес Vite с учетом нашего базового пути).
4. Сообщи пользователю, что локальный сервер запущен (если его пришлось запускать), и что сайт был успешно открыт в браузере по умолчанию.
