---
description: Обработка PDF файла (извлечение текста, картинок и скриншотов)
---
Этот скрипт извлекает текст, картинки и делает скриншоты каждой страницы из указанного PDF файла, чтобы я (Antigrave) мог их проанализировать.

// turbo
1. Запуск скрипта обработки
```bash
python3 scripts/pdf_processor.py "$1"
```
По умолчанию результаты будут сохранены в папку `processed_pdfs/`.
