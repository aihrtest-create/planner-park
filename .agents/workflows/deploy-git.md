# Выполнение автоматического деплоя на GitHub

Когда пользователь просит "Деплой на гит" или запускает этот воркфлоу, выполни следующие шаги, чтобы отправить локальные изменения в удаленный репозиторий.

// turbo-all
1. Выполни команду `git add .`, чтобы добавить все измененные или новые файлы в индекс.
2. Выполни коммит изменений с помощью команды `git commit -m "Автоматический деплой по запросу"`. 
3. Запушь коммит в ветку main с помощью команды `git push origin main`.
4. Сообщи пользователю об успешном деплое и напомни, что обновления появятся на сайте через минуту-две.
